# SHMTU(Shanghai Maritime University) Beamer Theme

#### Description

An unofficial LaTeX beamer template for Shanghai Maritime University students.

#### LaTeX Complier

XeLaTeX

#### Installation

1. Install TeXLive/macTeX
2. Install TeXstudio
3. Clone repo/Download zip
