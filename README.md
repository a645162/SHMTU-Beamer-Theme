#### LaTeX编译器

**XeLaTeX**

一定要使用XeLaTeX！！！否则卡你半天，最后报一堆错！

#### 安装教程

1. 安装 TeXLive/macTeX
2. 安装 TeXstudio(支持Windows/macOS/Linux)
3. 克隆仓库/下载ZIP

#### 使用说明

1. 设置TeXstudio的默认编译器为XeLaTeX
2. 修改内容
3. 编译tex
